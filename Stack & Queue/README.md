# Why we need Stack??
* 編譯器(word、sublime)的 undo 。
* 網頁瀏覽器中回到上一頁功能。
* 任何遞迴(recursion)形式的演算法都可以用 Stack 改寫，例如 Depth-First Search(DFS,深度優先搜尋)

# Stack function
* Push(Data): 把資料放到最上面(最新)。
* Pop: 把資料從最上面(最新)移除。
* Top: 回傳最上面(最新)的資料。
* IsEmpty: 確認stack 裡面是否有資料。
* getSize: 回傳stack 裡的資料個數。
